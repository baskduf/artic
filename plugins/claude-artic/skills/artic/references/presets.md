# Artic Presets

Presets are search hints and synthesis starting points, not rigid templates.

- clean-saas: minimal SaaS landing page, neutral surfaces, proof, clear CTA.
- developer-tool: code-forward, precise, docs-like, monospaced accents.
- ai-product: capability + trust, restrained gradients, workflow visuals.
- premium-consumer: Apple-like restraint without copying Apple identity.
- editorial-landing: narrative, type-led sections, magazine hierarchy.
- korean-startup: mobile-first, trust-heavy, Toss-like clarity without copying Toss.
- playful-brand: friendly rounded forms, color, light illustrations.
- luxury-minimal: sparse, high contrast, premium restraint.
