# Shopify Polaris Trust/Form Pattern Notes

## Reusable Patterns
- Prioritize trustworthy form flows, clear labels, and plain-language validation.
- Use calm surfaces, clear borders, and hierarchy that reduces user uncertainty.
- Extract commerce/admin UX principles only; do not copy Shopify brand assets or exact visuals.

## Components
- Form card with label, helper text, validation, and primary submit action.
- Trust proof close to conversion areas.
- Secondary actions should stay visually quieter than the primary conversion.

## Accessibility
- Form controls require labels, error copy, and contrast-safe states.
