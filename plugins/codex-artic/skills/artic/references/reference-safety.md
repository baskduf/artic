# Artic Reference Safety

Artic may study public and open-source design references, but it must not clone protected expression.

Allowed:
- Design token structure and naming ideas
- Color role relationships, not exact brand identity
- Typography hierarchy and rhythm
- Layout principles and spacing systems
- Component behavior patterns
- Accessibility rules
- Motion principles

Forbidden unless user owns rights:
- Logos and trademarks
- Proprietary illustrations
- Exact page compositions
- Exact branded color palettes as identity
- Exact copywriting or slogans
- Exact animation signatures
- Copyrighted imagery

Required generated-doc phrase:

Reference policy: extract reusable principles only; do not copy logos, trademarks, proprietary illustrations, or exact layouts.
