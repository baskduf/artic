# Artic DESIGN.md Output Contract

Required sections:
1. Overview
2. Colors
3. Typography
4. Layout
5. Elevation & Depth
6. Shapes
7. Components
8. Do's and Don'ts

Required front matter:
- version: alpha
- name
- description
- colors
- typography
- rounded
- spacing
- components

Use role-based token names: primary, secondary, accent, surface, neutral, text, muted, border.
Use token references in components, for example `{colors.primary}` and `{rounded.md}`.
Do not use reference brand names as token names.
