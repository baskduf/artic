# VoltAgent SaaS DESIGN.md Pattern Notes

## Reusable Patterns
- Lead with one concrete product promise and one primary conversion action.
- Use spacious sections, strong h1 contrast, and compact proof blocks.
- Treat brand-inspired examples as pattern references only; never copy exact layouts, names, copy, or palettes.

## Components
- Hero with primary CTA and secondary proof link.
- Feature cards grouped around user jobs.
- Social proof strip or metrics row after the hero.

## Accessibility
- Preserve readable body copy and clear focus/CTA hierarchy.
